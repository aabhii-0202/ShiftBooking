# ShiftBooking
## Installation
git clone https://github.com/aabhii-0202/ShiftBooking.git <br/>
cd ShiftBooking <br/>
change the baseUrl in the app/api/api.js as per your backend server <br/>
npm i <br/>
npm run android <br/>
